## shortcuts
* switch driver dir: `g, space` type d: `enter`
* `shift+enter`: open with(`enter`: open)
* `alt+o`: show openWith dialog
* `H`: go back in dir history
* `L`: go forward in dir history
## Commands
* `ya pkg upgrade`: update all installed plugins
* `ya pkg add <plugin-or-flavor>`
* `ya pkg remove <plugin-or-flavor-name>`
